Welcome to the EA-Explorer AddIn @ github.

The EA-Explorer Sparx Systems Enterprise Architect AddIn is a tool to inspect EAs Object Model API.

You'll get a selected API objects properties and collections presented in a standard property browser tool window. Currently the following object types will be supported:

  • otPackage (no collections yet)
  • otElement
  • otAttibute (no collections yet)
  • otMethod (no collections yet)
  • otConnector (no collections yet)

The tool is intended to aid Enterprise Architect automation and AddIn developers to see what information they'll get via the Object Model API, sampled from the current EA project.
